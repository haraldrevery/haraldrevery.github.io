Settings Tips

  • Show Preview — toggle the live rendered preview on or off. Hiding it gives you a full-width editor.
  • Show Outline — opens a heading navigator panel on the right. Click any heading to jump to it.
  • Reader Mode — hides the editor entirely for a clean, distraction-free reading view. Press Exit Reader Mode to return.
  • Mobile View — frames the preview at a phone-sized width so you can see how your content looks on small screens (don't rely on this too much...).
  • UI Size / Text Size — "UI Size" scales menu buttons; "Text Size" scales editor and preview text.
  • Calendar Format (Settings menu) — choose how dates are inserted when you use the date toolbar action.
  • Drag the divider — the vertical bar between editor and preview can be dragged left or right to resize each pane.
  • Click any preview block — jumps the editor cursor to the matching source line.
  • CPU performance delay — Higher value = Saves battery and CPU, but not that great experience. Low value = drains more CPU and battery but smoother experience.
  • Forced Prev. Synch. — "Forced Preview Synchronization" is a more reliable synchronization between the editor and preview window, but might feel a little janky. Use if you notice that the what you type is not visible on the preview.

Latex suppport

This editor has LaTeX support (through "KaTeX"). Use "$$ . . . $$" for equation blocks and "$ . . . $" for having the equation in the text. Note that after the dollar sign, no gap between the "$" and the first syntax symbol, otherwise it will not render it out correctly.
